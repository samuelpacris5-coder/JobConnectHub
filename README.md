## 🚀 Overview

**JobConnectHub** is a web-based job portal designed to simplify the hiring process by providing a centralized platform for job postings, applications, and candidate management.

It supports multiple user roles with dedicated dashboards and features.

---

## 👥 User Roles

### 🔑 Admin
- Manage users (Employers & Job Seekers)
- Approve or remove job postings
- Monitor platform activity
- View reports and analytics

---

### 🏢 Employer
- Create and manage job listings
- View and manage applicants
- Update company profile
- Track hiring progress

---

### 👨‍💻 Job Seeker
- Create and manage profile
- Upload resume/CV
- Search and apply for jobs
- Track application status

---

## ✨ Features

- 🔐 Multi-role authentication system  
- 📄 Job posting & management  
- 📬 Application tracking system  
- 🔍 Advanced job search & filters  
- 🧾 Resume upload support  
- 📊 Dashboard for each user role  
- 📢 Notifications (basic)  

---

## 🛠️ Tech Stack

- **Backend:** PHP  
- **Database:** MySQL
